# Library Manager


Small Python package for managing a library catalog with CLI and JSON persistence.


## Features
- `Book` class with convenience methods
- `LibraryInventory` with add/search/display and JSON persistence
- Menu-driven CLI at `cli/main.py`
- Basic logging and error handling
- Unit tests using `unittest`


## Quick start
1. Install Python 3.8+
2. Run the CLI:


```bash
python -m cli.main
